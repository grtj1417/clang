25. 計算面積 (積分)
本題使用切割面積加總法公式(梯形法)來計算函式圖形面積。

題目會選定一種函式，使用梯形法來估計函式 f(x) 的圖形面積，函式的 x 值從區間起始 p 到區間終點 q 間，也就代表著計算從 f(p) 到 f(q) 之間的面積，當估計出的面積值的差小於 10的負err次方時，則輸出"目前"的估計面積。

本題需計算的函式有兩種:
1. f1(x) = (a - x ^ 2) ^ (1 / 2)，a 為常數(整數或小數)
2. f2(x) = (a * (x ^ 3) + 7 * x) / (a + x) ^ (1 / 2)，a 為常數(整數或小數)

梯形法估計面積公式為:
A(n) = ((q - p) / (2 * n)) * [ f(p) + 2 * f(p + x) + 2 * f(p + 2x) + ... + 2 * f(q - x) + f(q) ]，其中 x = (q - p) / n

n 表示函數圖形被切割成的塊數，
初始設定 n = 1，每次計算出的精確度 err 若不符合所期待時，需將 n 放大兩倍 ( n = 2 * n)，再重新估計面積值。

err 表示精確小數位數，例如 err = 6，即 "目前"的切割與"上一次"的切割所估計出的面積值的差需小於 10 的負6次方，亦即小於 0.000001。

輸入說明:
此題測試資料為連續輸入一至多筆資料 (有可能一次輸入包含多組測資)。
第一行輸入整數 1 或 2， 1 代表計算 f1(x) 面積，2 代表計算 f2(x)面積，
第二行依序輸入 a, p, q, err，兩數字間以空白符號相隔開，a 表示函式中的常數，p 表示區間的起始位置，q 表示區間的終點位置，err 表示精確小數位數，
接下來
若輸入0，則結束輸入並結束程式
若輸入1，則代表計算f1(x)面積，且接下來重新輸入a, p, q, err，進行新一輪計算
若輸入2，則代表計算f2(x)面積，且接下來重新輸入a, p, q, err，進行新一輪計算


輸出說明:
根據每一組輸入，輸出"目前"的估計面積，答案需輸出到小數點第12位。
